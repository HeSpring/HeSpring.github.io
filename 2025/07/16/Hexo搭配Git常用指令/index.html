    前言

    为了防止我以后忘记了一键三连的指令，还有方便以后的学习，搜索总结了这一篇文章。

    一、Hexo 基本操作指令

    操作 命令
    初始化部署插件（只需要一次） npm install hexo-deployer-git --save
    初始化 Hexo 项目 hexo init
    安装依赖 npm install
    启动本地预览服务器 hexo s 或 hexo server
    清理缓存 hexo clean
    生成静态页面 hexo g 或 hexo generate
    部署网站（需要配置 _config.yml 中的 deploy） hexo d 或 hexo deploy
    一键生成并部署 hexo g -d

    二、新建内容相关命令

    操作 命令
    新建文章 hexo new "文章标题"
    新建页面（如关于页、留言板） hexo new page "about"
    新建草稿（需要 _config.yml 配置 post_asset_folder: true） hexo new draft "草稿标题"
    发布草稿为正式文章 hexo publish "草稿标题"

    四、常见 Git 清理与回滚命令

    操作 命令
    查看状态 git status
    查看历史 git log
    撤销某次 commit git reset --soft HEAD^（保留修改）
    强制回退到某个版本 git reset --hard 提交ID
    丢弃所有未提交改动 git checkout .
    强推（不推荐，谨慎使用） git push origin main --force
