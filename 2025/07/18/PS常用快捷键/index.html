PS常用快捷键

发表于 2025-07-18 更新于 2025-07-18 分类于 摄影 阅读次数： 本文字数： 788 阅读时长 ≈ 3 分钟

Photoshop 2021（PS2021）常用快捷键大全，覆盖了工具切换、图层操作、选择、编辑等多个方面，适合日常使用和高效工作。

工具快捷键

快捷键 功能
V 移动工具
M 矩形/椭圆选框工具
L 套索工具（按Shift循环多种）
W 快速选择工具 / 魔棒工具
C 裁剪工具
I 吸管工具
J 修复画笔/污点修复等
B 画笔工具
S 仿制图章工具
E 橡皮擦工具
G 油漆桶/渐变工具
A 路径选择工具
T 文字工具
P 钢笔工具
U 形状工具
H 抓手工具
Z 缩放工具
D 恢复默认前景/背景色
X 交换前景/背景色

图层相关

快捷键 功能
Ctrl + J 复制图层
Ctrl + Shift + J 剪切图层
Ctrl + G 组合图层为组
Ctrl + Shift + G 取消图层组
Ctrl + E 合并选中图层
Ctrl + Shift + E 合并可见图层
Ctrl + Alt + Shift + E 盖印图层（新建合并图层）
Ctrl + T 自由变换
Ctrl + [ / ] 向下/上移动图层顺序
Ctrl + Alt + [ / ] 置于底层 / 顶层

编辑操作

快捷键 功能
Ctrl + Z 撤销（多步）
Ctrl + Alt + Z 多步撤销
Ctrl + Shift + Z 重做
Ctrl + C / V / X 复制 / 粘贴 / 剪切
Ctrl + A 全选
Ctrl + D 取消选择
Ctrl + F 应用上一个滤镜
Ctrl + Shift + I 反选
Ctrl + Backspace 填充背景色
Alt + Backspace 填充前景色

视图控制

快捷键 功能
Ctrl + + / - 放大 / 缩小
Ctrl + 0 屏幕适配大小
Ctrl + 1 实际像素显示
空格 + 拖动 临时切换抓手工具
Tab 显示/隐藏面板
F 切换屏幕模式

文件操作

快捷键 功能
Ctrl + N 新建文件
Ctrl + O 打开文件
Ctrl + S 保存文件
Ctrl + Shift + S 另存为
Ctrl + Alt + Shift + S 存储为Web（导出）

常用辅助操作

快捷键 功能
Ctrl + R 显示/隐藏标尺
Ctrl + ; 显示/隐藏参考线
Ctrl + ’ 显示/隐藏网格
Shift + 拖动对象 保持比例
Alt + 拖动对象 快速复制
Ctrl + 鼠标点击图层缩略图 载入选区
