NX007常用快捷键

发表于 2025-07-18 更新于 2025-07-23 分类于 机械 阅读次数： 本文字数： 840 阅读时长 ≈ 3 分钟

Siemens NX 2007（也称 NX 2207，即 NX 的 2022 年 7 月版本）中一些常用的快捷键列表，适用于建模、装配、制图等模块：

通用快捷键

快捷键 功能描述
Ctrl + N 新建文件
Ctrl + O 打开文件
Ctrl + S 保存文件
Ctrl + P 打印
Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切
Ctrl + A 全选
Ctrl + Tab 切换窗口

建模模块快捷键

快捷键 功能描述
Ctrl + Shift + E 拉伸（Extrude）
Ctrl + Shift + R 旋转（Revolve）
Ctrl + Shift + H 孔（Hole）
Ctrl + Shift + P 拔模（Draft）
Ctrl + Shift + U 联集（Boolean Unite）
Ctrl + Shift + D 差集（Boolean Subtract）
Ctrl + Shift + I 插入（Boolean Intersect）
Ctrl + Shift + F 倒圆角（Fillet）
Ctrl + Shift + C 倒斜角（Chamfer）

草图（Sketch）快捷键

快捷键 功能描述
L 线段 Line
R 矩形 Rectangle
C 圆 Circle
S 修剪 Trim
A 对齐 Align
T 偏置 Offset
D 尺寸 Dimension
E 等距 Equal

装配模块快捷键】

快捷键 功能描述
Ctrl + Shift + M 添加组件
Ctrl + Shift + A 装配约束（Assembly Constraint）
Ctrl + Shift + S 替换组件
Ctrl + Shift + G 爆炸视图生成

制图模块（Drafting）快捷键

快捷键 功能描述
V 投影视图 View
B 剖视图 Section
P 部分视图 Partial View
D 尺寸 Dimension
T 文本 Text
C 中心线 Centerline

查看与导航快捷键

快捷键 功能描述
F8 动态旋转
F9 动态缩放
F10 动态平移
Ctrl + 1 ~ Ctrl + 6 六视图快速切换
Ctrl + Shift + Q 设置工作坐标系

提示：

  • NX 支持自定义快捷键，可通过 菜单 > 工具 > 快捷键（Customize） 进行修改。
  • 部分快捷键可能因版本或界面语言不同略有变化。
