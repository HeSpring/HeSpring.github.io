发表于 2025-09-10 更新于 2025-09-10 分类于 Hexo 阅读次数： 本文字数： 19k 阅读时长 ≈ 1:05

前言

在个人博客中，如果你希望展示一组精美图片或者作品集，一个带有分类筛选、搜索功能、分页以及点击放大预览的相册页面会非常吸引人。本文分享我在 Hexo + NexT 博客中实现的影画册页面。完整代码在最后。

一、主要功能

该影画册页面实现了以下功能：

  1. 主分类按钮：Galgame、二游、动漫、风景、插图等

  2. 子分类按钮：如“婚纱”“亲吻”“偶像”等，点击主分类动态显示

  3. 搜索功能：输入图片名称即可快速过滤

  4. 分页展示：每页显示 9 张图片，自动生成分页按钮

  5. 点击放大预览：

    • 图片点击显示大图

    • 左右箭头支持切换

    • 二次元风格箭头：

      • 默认半透明淡粉色，带发光

      • 悬停时放大，光效增强

      • 点击时缩小，模拟按下效果
